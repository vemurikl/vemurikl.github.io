Course Overview - Topics in Computer Systems (001)

Course Instructor: Phu Phung

Course Summary

In this course, students will study basic web application development with front-end (HTML5, JavaScript, CSS) and back-end (PHP/MySQL). Secure programming principles and practices will be introduced to avoid potential web application vulnerabilities and attacks.

Course Activities

- Lab 1: Introduction to HTML5 and CSS
- Lab 2: JavaScript Essentials
- Hackathon: Secure Web Application Development

Overview

This course provides an in-depth understanding of web application development and security. Students will learn about client-side and server-side technologies, version control, and secure programming principles. The course emphasizes hands-on exercises, labs, and a hackathon to apply theoretical knowledge in practical scenarios.

Projects

Evaluation Of Data Analytical Tools

Studied and tested popular data tools like Excel, Tableau, Power BI, and ApacheSpark. Made charts showing what each tool can do well versus not as well. Looked at how easy they are to use, if they work with large amounts of data, and how they work together. Wrote a report of findings to help company executives pick the best data tools for different needs in the business.

Prediction Of Hand Gestures for Impaired Persons using OpenCV Approach

Developed hand gesture recognition system using OpenCV and deep learning. Classified hand gestures through modeling and algorithm optimization. Enables natural, user-friendly communication interface for hearing impaired. Demonstrates the potential of computer vision and ML for assistive technology.
